    프로젝트 진척율

    2. 일정변경 관리하지 않을 때

    프로젝트 작업량과 진척율을 보여줍니다. "스케쥴 업데이트" 버튼 클릭 시 아래 항목이 자동 계산됩니다.

    시작일과 완료일이 선택되지 않은 데이터는 진행율 계산에서 제외됩니다.

    Man/Day 프로젝트 전체의 작업량을 Man/Day로 표시한다. 가중치를 적용한 값이다.
    Man/Day 해당작업의 시작일부터 현재일까지의 작업기간을 의미함(현재 날짜까지 진행되었어야 할 기간)
    Man/Day 실적기성(총기간 x 총실적진행율)
    % 현재날짜까지 계획된 진척율, WBS의 "계획진척율"의 전체 합계 진척율을 표시함
    % 현재날짜까지 실제로 진행된 진척율, WBS의 "실적진척율"의 전체 합계 진척율을 표시함
    % 양수이면 계획보다 빨리 진행, 음수이면 지연
